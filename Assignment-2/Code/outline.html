Course Modules

Module Title Description Details
I Fundamentals of Compiler Overview - Structure of a compiler, applications of compiler, one pass and two pass compiler.
Lexical analysis - The role of a lexical analyzer, specification of tokens, recognition of tokens, LEX.

Detailed Information:

In this module, you'll explore the structure of compilers in-depth, understand various types of compilers, and study lexical analysis techniques in detail.
II Syntax Analysis Context-free grammar, writing grammars for context-free environments, parse trees and ambiguity, role of parser.
Specification and recognition of tokens, top-down parsing, recursive descent and predictive parsers (LL), bottom-up parsing, operator precedence parsing, LR, SLR, and LALR parsers.

Detailed Information:

This module covers context-free grammars, parse trees, and various parsing techniques, including top-down and bottom-up parsing.
III Syntax Directed Translation & Run Time Environments Syntax-directed definitions, evaluation orders for attributes of an SDD, S-attributed and L-attributed SDDs.
Construction of syntax tree, source language issues, storage organization and allocation strategies, parameter passing.
Symbol table organizations and generations, dynamic storage allocations.

Detailed Information:

This module delves into syntax-directed translations, runtime environments, and symbol table management.

Assignment statements and Boolean expressions, case statements, back patching, procedure calls.

Detailed Information:

This module focuses on intermediate code generation techniques and representations.
V Code Optimization Sources of optimization, basic blocks and flow graphs, optimization of basic blocks, loops in flow graphs.
Loop optimization, machine-independent optimization, machine-dependent optimization, dead-code elimination, code improving transformations.

Detailed Information:

This module covers various code optimization techniques to enhance performance.
VI Code Generation Issues in the design of a code generator, run time storage management.
Simple code generator - register and address descriptors, code generation algorithm, design of the function getReg, DAG, peephole optimization.
Register allocation and assignment, selection of instruction, register allocation, parallel compilation, Just-in-Time compiler, study of compiler construction tools.

Detailed Information:

This module addresses code generation strategies and runtime management.
